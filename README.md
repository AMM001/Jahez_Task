# 🎬 Movies App (SwiftUI + SPM)

A modular iOS Movies application built using **SwiftUI**, **Combine**, and **MVVM architecture**, with a strong focus on scalability and separation of concerns using **Swift Package Manager (SPM)**.

---

## 📦 Project Structure

This project is divided into **two Swift Packages (SPM):**

### 1️⃣ Movies-SPM
Responsible for the **presentation layer**.

#### Features:
- 🎞 Movies List Screen
- 🔍 Search functionality
- 🎭 Filter by genres
- 📄 Movie Details Screen
- 🔄 Pagination (Load More)
- ⏳ Loading states handling

#### Architecture:
- MVVM (Model - View - ViewModel)
- SwiftUI for UI
- Combine for reactive bindings

---

### 2️⃣ NetworkLayer-SPM
Handles all **networking logic**.

#### Responsibilities:
- 🌐 API requests handling
- 🔗 Endpoint management
- 📦 Response decoding
- ⚠️ Error handling

#### Current State:
- ✅ Implemented as a **Local SPM**
- 🚧 **Phase 2:** Will be published as a **Remote Package**

---

## 🧱 Tech Stack

- **SwiftUI** → UI Development
- **Combine** → Reactive Programming
- **MVVM** → Clean Architecture
- **SPM (Swift Package Manager)** → Modularization
- **AsyncImage** → Image loading
